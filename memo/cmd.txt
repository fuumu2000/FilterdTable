1. **コンポーネント名**：FilterdTableComponent
2. **機能**：
・入力データをもとに各項目をフィルターするために、ユニークなデータとして取り出す、その時”全データ”という文字列を追加
・filterswがTrueの場合にはその項目の右端にMUIICONでフィルター項目を表示するアイコンを表示
・アイコンをクリックされたら選択候補を表示
・選択されたらその項目用のフィルターとして保存、なお全データが選択された場合はその項目のフィルターはクリア、選択後はチェックを付加してアイコンをクリックした際にわかるようにする
・すべてのアイコンが付加されている項目のフィルターを複合条件としてフィルターする
3. **Props**：以下の型
export type tblinput = {
  tbltitle: string[];   // tableのタイトル行のデータ
  tbldata: string[][];  // 表示するデータ
  filtersw: boolean[];  // 各項目にフィルターボタンを付加するかどうか
　height: number;       // データを表示する領域の縦のピクセル数、省略時は250
};
4. **State**：各項目のフィルターの選択済みの値
